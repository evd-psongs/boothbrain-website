## üéØ **PRO TIPS FOR GREAT SCREENSHOTS**

### Before Taking Screenshots:
1. **Add product photos** - Use free stock photos from:
   - Unsplash.com (free, no attribution)
   - Pexels.com (free, no attribution)
   - Pixabay.com (free, no attribution)

2. **Use realistic photos:**
   - Search: "handmade jewelry", "candles", "honey jar", etc.
   - Download high-quality (1080p+)
   - Upload to inventory items

3. **Create visual variety:**
   - Mix of colors in product photos
   - Different price ranges visible
   - Various stock levels (low stock = 5, high stock = 45)

4. **Perfect timing:**
   - Set active session to show mid-day sales (around noon)
   - Show progress (10 sales, $861 revenue looks active but not overwhelming)

### During Screenshots:
1. Hold phone in portrait mode
2. Clean interface (no debug popups)
3. Take 2-3 shots of each screen (pick best later)
4. Show realistic scrolling (not top of long lists)

### Screenshot Checklist:
- [ ] All products have photos (no placeholders)
- [ ] Events have realistic names/dates
- [ ] Session shows active sales
- [ ] Analytics has data to display
- [ ] No "test" or "lorem ipsum" text
- [ ] UI is in light mode
- [ ] Time shows 9:41 AM
- [ ] Full battery/signal strength
