## üì± App Store Requirements

### iOS (Apple App Store)
- **Icon:** 1024x1024px (no transparency, no rounded corners)
- **Screenshots:** 3-10 per device size (iPhone 6.7", 6.5", iPad 12.9")
- **Privacy URL:** https://boothbrain.app/privacy (must be live)
- **Support URL:** https://boothbrain.app/support (must be live)
- **Age Rating:** 4+

### Android (Google Play)
- **Icon:** 512x512px
- **Feature Graphic:** 1024x500px
- **Screenshots:** 2-8 phone screenshots
- **Privacy URL:** https://boothbrain.app/privacy (must be live)
- **Support Email:** song.sopaul@gmail.com (must be active)
- **Content Rating:** Everyone
